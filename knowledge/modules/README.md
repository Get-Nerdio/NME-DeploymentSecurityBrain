# Secondary Modules

Expansion slot (Phase 4). One subdirectory per secondary module, e.g.:

```
modules/
  <module-name>/
    overview.md
    permissions.md   # links specific rows to ../../permissions/permission-matrix.md
```

Each module's permissions must be reflected in the canonical
[permission-matrix.md](../permissions/permission-matrix.md) (tagged with the owning module),
not stated only here.
